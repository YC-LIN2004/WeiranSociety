# Backend — WeiranSociety（未然學舍）

本資料夾為WeiranSociety的後端系統(Spring Boot)
提供所有API、資料庫存取、金流處理、權限驗證、角色管理等核心功能。

後端採用 Java 17 + Spring Boot 3，搭配 MSSQL 作為主要資料庫，  
並整合 JWT、Email、File Upload、ECPay 金流等服務。

#技術架構（Tech Stack）
Java 17
Spring Boot 3
Spring Security（JWT 驗證與授權）
Spring Data JPA / Hibernate
Microsoft SQL Server（MSSQL）
ECPay 綠界金流（Stage 測試環境）
Gmail SMTP（Email）
Maven Wrapper（mvnw）
